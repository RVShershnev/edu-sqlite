Где использовать SQLite:

* Встроенные устройства и Интернет вещей  
* Файловый формат данных   
* Веб-сайты (при трафике менее 100 000 запросов в день)  
* Кэш данных
* Замена файлов на диске (файл базы данных ограничен размером 281 терабайт)
* Для тестирования

Где не нужно использовать SQLite:

* Клиент и файл базы данных находится на разных устройствах
* Много конкуретных писателей
* Большие данные
